# School Library

This project is a database for a school library. It can keep track of the record of books. It is build with SQL and PsotgreSQL.

## 💻 Getting Started <a name="getting-started"></a>

> This guide will walk you through the steps to get the project up and running on your local machine.

### Prerequisites
In order to run this project you need: to dowload it or clone it to your machine.

### Setup


1- Clone the repository to your local machine using the following command:

git clone https://github.com/mohamedgamaleldin999999/school-library

2- Change into the project directory:

cd [project directory]

3- Install the required dependencies:

npm install
